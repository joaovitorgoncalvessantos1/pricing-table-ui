# 💳 Pricing Table UI

## 📖 Sobre o projeto
Este é um projeto de interface de tabela de preços (pricing table), desenvolvido com foco em design moderno e prática de HTML e CSS.

A proposta foi criar um componente visual utilizado em sites reais para apresentar planos, preços e benefícios de forma clara e organizada.

---

## ✨ Funcionalidades
- Exibição de planos (Essencial, Premium, etc.)
- Comparação de preços
- Lista de benefícios por plano
- Botões de ação (Assine já)
- Layout organizado em cards

---

## 🎨 Layout
O projeto possui:
- Design moderno e minimalista
- Cards alinhados e responsivos
- Destaque visual para planos
- Interface inspirada em sites de serviços e SaaS

---

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3

---

## 🎯 Aprendizados
Durante esse projeto, pratiquei:
- Criação de componentes reutilizáveis
- Organização de layout com CSS
- Alinhamento e espaçamento de elementos
- Construção de interfaces inspiradas em produtos reais
